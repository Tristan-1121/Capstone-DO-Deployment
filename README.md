Instructions for Running the App:
You will need two bash terminals for this.

Terminal One:
CD into the UWF-CareConnect directory
Enter "npm i"
Enter "npm run server"

Terminal Two:
CD into frontend directory from UWF-CareConnect
Enter "npm i"
Enter "npm run dev"

This should bring the website up and allow you to use it.

In case it doesn't work, please try npm i mongoDb and/or npm i cors
